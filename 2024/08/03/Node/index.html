Stream

可写流：fs.createWriteStream()
可读流：fs.createReadStream()
双工流 转换流
应用：

 if (method === 'GET') { // get 请求
  const fileName = path.resolve(__dirname, 'data.txt');
  let stream = fs.createReadStream(fileName);
  stream.pipe(res); // 将 res 作为 stream 的 dest
}

// 两个文件名
const fileName1 = path.resolve(__dirname, 'data.txt')
const fileName2 = path.resolve(__dirname, 'data-bak.txt')
// 读取文件的 stream 对象
const readStream = fs.createReadStream(fileName1)
// 写入文件的 stream 对象
const writeStream = fs.createWriteStream(fileName2)
// 通过 pipe执行拷贝，数据流转
readStream.pipe(writeStream)
// 数据读取完成监听，即拷贝完成
readStream.on('end', function () {
    console.log('拷贝完成')
})

框架相关（koa exp）

koa是基于NodeJS当前比较流行的web框架，本身支持的功能并不多，功能都可以通过中间件拓展实现。通过添加不同的中间件，实现不同的需求，从而构建一个 Koa 应用

中间件参数：
ctx: 封装了request response的变量
next：进入下一个执行的中间件

示例：

const fs = require('fs')
module.exports = (options) => async (ctx, next) => {
  const startTime = Date.now()
  const requestTime = new Date()
  await next()
  const ms = Date.now() - startTime;
  let logout = `${ctx.request.ip} -- ${requestTime} -- ${ctx.method} -- ${ctx.url} -- ${ms}ms`;
  // 输出日志文件
  fs.appendFileSync('./log.txt', logout + '\n')
}

内置的中间件： koa-bodyparser(转换post请求，挂载ctx.request上)
koa-static(处理静态资源)等
